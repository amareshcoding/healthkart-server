# healthkart-server

## Run Locally

Clone the project

```bash
  git clone https://github.com/amareshcoding/healthkart-server
```

Go to the project directory

```bash
  cd healthkart-server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
